# Dev - FE - NPM

## NPM \(package.json\)

Following are the targets in `package.json`

| `Target Name` | Description |
| :--- | :--- |
| ng | Run Angular CLIs ng command |
| copy | Copy models in to FE codebase |
| serve | Copy models and run Angular's ng serve |
| build | Copy models and run Angular's ng build |
| test | Copy models and run Angular's ng test |
| e2e | Copy models and run Angular's ng e2e |
