# Dev - Requrements

Following is a list of prerequisite tools / frameworks to be installed before installation is attempted

* Nodejs 12+

Following are the fundamental tools / frameworks that are being used in the application. 

* Typescript
* Angular 9+
* Nodejs 12+
* Angular Materials
* Angular Flex Layout
* Nodemailer
* SASS
* ExpressJs
* MariaDB
