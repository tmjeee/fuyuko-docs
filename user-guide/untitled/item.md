# Item

Items can be listed in three formats

* Tabular
* Thumbnail
* List

## 1\) Tabular

This is the tabular representation of Item. Each item will have the following mandatory fields :-

* Name, unique across view
* Description
* Images, where only one can be marked as primary.
* Attributes, at least one.

### Tabular Listing

* Click on item's name, description or attribute will popup a dialog allowing them to be edited, 
* To delete multiple items, check the appropriate checkboxes and then click on the "Delete" button
* Clicking on the "Add" button will result in a new row being added at the bottom of the table, allowing user to edit each fields and attribute individually.
* Click on the "+" icon on the far left to add an Item, where an empty child row will be created, allowing the user to edit each fields and attributes individually.
* Click on the "bin" icon on the far left to delete that item
* Click on the "image" icon on the far left to popup a dialog where one can :-
  * View images of the item
  * Delete Image of the item
  * Mark Image as primary

![](../../.gitbook/assets/item-tabular-list.png)

### Listing with children expanded

Item is hierarchical, they can have children, the same applies to its children.

![](../../.gitbook/assets/item-tabular-list-with-children.png)

### Searching

Item table is searchable. 

{% hint style="info" %}
"Search Item" text field is not auto-complete capable, user will need to type the search term and press enter for searching to take place.
{% endhint %}

![](../../.gitbook/assets/item-tabular-search.png)

### Filtering

Click on the "Filter" button will expand a filtering section. In this section user can:

* filter in or out which attribute to show in the table by checking or unchecking the checkboxes correspoding to the attribute respectively
* Move attribute up and down dictating the order they will be show in the table where being in the top will allow them to be shown on the left and otherwise.

![](../../.gitbook/assets/item-tabular-filter.png)

### Adding New Items

* Clicking on the "+" icon on individual item will result in a new row being created under the item, representing the child item of that item, to be created
* Clickin on the "Add" button will result in  a blank row, representing a new item, being created at the bottom of the table.

![](../../.gitbook/assets/item-tabular-add-item.png)

### Edit Item Image

Clicking on the "Image" icon on an item will popup a dialog allowing:

* Deleteing an image \(the "bin" icon, first icon left bottom of the image carousel\)
* Marking an image as primary \( the "bookmark" icon, second icon left bottom of the image carousel\)
* Upload a new image \(the "upload" icon, third icon left bottom of the image carousel\) 

![](../../.gitbook/assets/item-tabular-edit-item-image.png)
